## 当前包含插件

### 超链接悬浮增加红色下划线
```bash
不限页面。
鼠标悬浮在页面内的超链接上时增加红色下划线，并在移出时恢复初始状态。
PS：如果超链接本身存在下划线，也会移除
```

### 右键清除浏览器缓存
```bash
不限页面。
Cookie仅清除当前网站的所有缓存。
```
